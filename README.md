# GradFinale

## Prerequisite

Install `bun` globally:

```bash
npm install -g bun
```

Install the dependencies:

```bash
bun install
```

## Run the project

To run both the backend and the frontend, use the following command:

```bash
bun run dev
# OR
npm run dev
```

To run just the backend:

```bash
bun run dev-ws
# OR
npm run dev-ws
```

To run just the frontend:

```bash
bun run dev-app
# OR
npm run dev-app
```
